## Usage

### Base

[![img-npm-base]][lnk-npm-base]

```sh
npm i -DE @ivangabriele/tsconfig-base
```

or:

```sh
yarn add -DE @ivangabriele/tsconfig-base
```

Then extend it in your `tsconfig.json`:

```json
{
  "extends": "@ivangabriele/tsconfig-base"
}
```
